з”Ё canvas еҒҡиҝҮйңҖжұӮзҡ„жңӢеҸӢеӨҡе°‘еә”иҜҘеҶҷиҝҮ ctx.beginPath ж–№жі•пјҢдёҖж—ҰжҹҗдёӘдҪҚзҪ®е°‘дәҶиҝҷдёӘж–№жі•зҡ„и°ғз”ЁпјҢcanvas з»ҳеҲ¶е°ұеҸҜиғҪеҮәзҺ°йў„жңҹд№ӢеӨ–зҡ„ж•ҲжһңгҖӮдҫқзЁҖи®°еҫ—дёҖе№ҙеүҚеҲқеӯҰ canvas зҡ„ж—¶еҖҷжңүдёҖдёӘеҜ№ canvas зҡ„жҸҸиҝ°вҖ”вҖ”вҖңеҹәдәҺзҠ¶жҖҒзҡ„з»ҳеӣҫвҖқпјҢеҪ“ж—¶дјјжҮӮйқһжҮӮпјҢзӣҙеҲ°жңҖиҝ‘еҶҷдёҡеҠЎеҪ»еә•жҗһжё…жҘҡз„¶еҗҺжқҘз®ҖеҚ•еҲҶдә«дёҖдёӢгҖӮ

еҰӮжһң

  1. дҪ жғіе…Ҙй—Ё canvasпјҢе®Ҹи§ӮдёҠеҜ№ canvas зҡ„з»ҳеҲ¶жөҒзЁӢжңүдёҖдёӘжё…жҷ°зҡ„жҠҠжҸЎ
  2. дҪ дёҚжҳҜеҫҲжё…жҘҡз©¶з«ҹ beginPath() зҡ„дҪңз”Ёд»ҘеҸҠдҪ•дёәвҖңcanvas жҳҜеҹәдәҺзҠ¶жҖҒзҡ„з»ҳеӣҫвҖқ

еҸҜд»ҘзңӢдёӢж–Үз« 

canvas з»ҳеӣҫе…Ҙй—ЁдёҺзҗҶи§Ј вҖӢ

е’ұд»¬дёҚеҺ»з»ҶиҜҙ apiпјҢд»Һе®Ҹи§ӮдёҠпјҢжҲ‘и°ҲдёҖдёӢжҲ‘еҜ№ canvas з»ҳеӣҫзҡ„зҗҶи§ЈгҖӮ

з»ҷдёҚеӨӘзҶҹжӮү canvas зҡ„жңӢеҸӢд»¬еӨҡиҜҙдёҖдәӣпјҡжҲ‘д»¬еңЁз»ҳеҲ¶ canvas ж—¶йңҖиҰҒе…ҲиҺ·еҸ– canvas е…ғзҙ еҜ№еә”зҡ„дёҠдёӢж–ҮеҜ№иұЎпјҢз„¶еҗҺйҖҡиҝҮдёҠдёӢж–ҮеҜ№иұЎеҺ»з»ҳеҲ¶еҜ№еә”зҡ„ canvasпјҢеҰӮдёӢйқўзҡ„ж —еӯҗпјҡ

vue
<canvas id="demo"></canvas>
<script>
const canvasElm = document.getElementById('demo');
canvasElm.width = 300;
canvasElm.height = 300;
canvasElm.style.border = `1px solid red`;
вҖӢ
// еҲӣе»ә canvas дёҠдёӢж–Ү
const ctx = canvasElm.getContext('2d');
</script>

ж—ўз„¶д»Јз ҒеҶҷеҮәжқҘдәҶпјҢжҲ‘иҝҳжҳҜеӨҡиҜҙдёӨеҸҘпјҢжҠҠд»Јз Ғи§ЈйҮҠжё…жҘҡпјҡеңЁ js дёӯжӢҝеҲ° <canvas/> ж Үзӯҫзҡ„еҜ№еә” domcanvasElm еҗҺпјҢйҖҡиҝҮ canvasElm.width / height жқҘи®ҫзҪ®зҡ„е®Ҫй«ҳе…¶е®һе№¶дёҚжҳҜзңҹжӯЈж„Ҹд№үдёҠзҡ„и®ҫзҪ® dom зҡ„е®Ҫй«ҳж ·ејҸпјҢиҝҷйҮҢзұ»дјјдәҺз”іиҜ· canvas з”»еёғзҡ„еғҸзҙ зӮ№пјҢеҸӘжҳҜиҜҙеңЁдёҚжҳҫејҸи®ҫзҪ® canvas зҡ„ css ж ·ејҸж—¶ canvas еғҸзҙ зӮ№еӨ§е°ҸзӯүдәҺдёҖдёӘ css еғҸзҙ еӨ§е°ҸгҖӮд№ҹе°ұжҳҜиҜҙз»ҸиҝҮиөӢеҖј canvasElm.width / height еҗҺжҲ‘д»¬зҡ„ canvas жҳҜдёҖеј  300 * 300 зҡ„з”»еёғпјҲз”ЁдәҺз»ҳеҲ¶е®ҡдҪҚпјүгҖӮе№¶дё”жҲ‘д»¬жІЎжңүжҳҫејҸи®ҫзҪ® canvas зҡ„ css е®Ҫй«ҳпјҢжүҖд»Ҙ canvas зҡ„е®Ҫй«ҳе°ұй»ҳи®Өд№ҹзӯүдәҺ 300px * 300px дәҶгҖӮд»ҺиҝҷйҮҢеҸҜд»Ҙе»¶дјёеҲ°е…ідәҺ canvas з”»иҙЁдјҳеҢ–зҡ„дёҖдёӘжүӢж®өпјҢж— е…ідё»йўҳе’ұе°ұдёҚеӨҡиҜҙдәҶгҖӮз„¶еҗҺ canvasElm.getContext('2d') иҝ”еӣһдәҶдёҖдёӘ canvas дёҠдёӢж–ҮеҜ№иұЎпјҢи°ғз”ЁиҝҷдёӘеҜ№иұЎдёҠзҡ„ api е°ұеҸҜд»Ҙе®ҢжҲҗеҜ№ <canvas id="demo"> зҡ„з»ҳеҲ¶гҖӮ

canvas з»ҳеҲ¶зҡ„иҝҮзЁӢжҳҜе…Ҳжһ„жҖқеҶҚеҠЁз¬”зҡ„иҝҮзЁӢгҖӮ иҝҷйҮҢзҡ„жһ„жҖқе№¶дёҚжҳҜжҢҮжҲ‘д»¬еӨ§и„‘йҮҢзҡ„жһ„жҖқпјҢиҖҢжҳҜиҜҙ canvas дёҠдёӢж–ҮдёҠжҸҗдҫӣзҡ„з»ҳеҲ¶зӣёе…ізҡ„ api еӨ§дҪ“еҸҜд»ҘеҲҶдёәдёӨз§ҚпјҢдёҖз§ҚжҳҜвҖңжһ„жҖқ apiвҖқпјҢеҸҰдёҖзұ»е°ұжҳҜвҖңеҠЁз¬” apiвҖқпјҢзңӢдёӢйқўзҡ„ demo з§’жҮӮпјҡ

js
const ctx = canvasElm.getContext('2d');
ctx.moveTo(0, 0); // иҝҷйҮҢзҡ„ 0, 0 е’ҢдёӢйқўзҡ„ 150, 150 йғҪжҳҜй’ҲеҜ№дёҠйқўжҲ‘д»¬жүҖиҜҙзҡ„ canvas з”»еёғеӨ§е°ҸпјҢcanvas з”»еёғе·ҰдёҠи§’дёә (0, 0) зӮ№пјҢеҗ‘еҸідёә x иҪҙпјҢеҗ‘дёӢдёә y иҪҙгҖӮ
ctx.lineTo(150, 150);

moveTo дёҺ lineTo е°ұжҳҜвҖңжһ„жҖқ apiвҖқпјҢйЎҫеҗҚжҖқд№ү moveTo дёҺ lineTo е°ұжҳҜз§»еҠЁзҡ„ж„ҸжҖқпјҢд»–д»¬еңЁжҸҸиҝ°дёҖдёӘи·Ҝеҫ„пјҢmoveTo жҢҮз§»еҠЁеҲ°жҹҗдёӘе…·дҪ“зҡ„зӮ№пјҢlineTo е°ұжҳҜеҹәдәҺзҺ°еңЁзҡ„дҪҚзҪ®дёҺзӣ®ж ҮдҪҚзҪ®з”»дёҖжқЎзәҝгҖӮдҪҶжҳҜд»–д»¬е№¶дёҚдјҡзңҹжӯЈзҡ„иҝӣиЎҢз»ҳеҲ¶пјҢд№ҹе°ұжҳҜиҜҙз»ҸиҝҮдёҠйқўзҡ„ж“ҚдҪңпјҢcanvas дёҠе№¶жІЎжңүзңҹжӯЈзҡ„еҮәзҺ°з»ҳеҲ¶зҡ„зәҝжқЎпјҢжҲ‘д»¬жҠҠ moveTo дёҺ lineTo дә§з”ҹзҡ„з—•иҝ№зҗҶи§ЈдёәдёҖз§Қжһ„жҖқгҖӮжңүдәҶвҖңжһ„жҖқвҖқпјҢзҺ°еңЁе°ұйңҖиҰҒвҖңеҠЁз¬” apiвҖқжқҘе°Ҷжһ„жҖқвҖңеҸҳзҺ°вҖқгҖӮ

js
const ctx = canvasElm.getContext('2d');
ctx.moveTo(0, 0);
ctx.lineTo(150, 150);
// еҠЁз¬” apiвҖ”вҖ”stroke
ctx.stroke();

жүҖи°“ strokeпјҢпјҲиҪ»жҠҡпјӣж‘©пјӣжҚӢпјӣеңЁвҖҰ дёҠиҪ»иҪ»ж¶ӮжҠ№пјүпјҢдҪҶдҪңз”Ёе°ұжҳҜиҝһзәҝпјҢе°ҶвҖңжһ„жҖқвҖқдә§з”ҹзҡ„зәҝжқЎзңҹжӯЈзҡ„з”»еңЁ canvas дёҠгҖӮз»ҸиҝҮдёҠйқўвҖңжһ„жҖқвҖқ + вҖңеҠЁз¬”вҖқзҡ„ж“ҚдҪңпјҢжҲ‘д»¬зҡ„ canvas дёҠе°ұжңүдәҶдёҖжқЎд»Һе·ҰдёҠи§’еҲ°дёӯеҝғзҡ„й»‘зәҝгҖӮ

зҗҶи§ЈеҹәдәҺзҠ¶жҖҒзҡ„з»ҳеӣҫ вҖӢ

е…ҲзңӢдёҖж®өд»Јз Ғз»“еҗҲз»ҳеҲ¶ж•Ҳжһңж„ҹеҸ—дёҖдёӢпјҡ

js
const ctx = canvasElm.getContext('2d');
ctx.moveTo(0, 0);
ctx.lineTo(150, 150);
ctx.stroke(); // й»ҳи®Өиҝһзәҝзҡ„йўңиүІжҳҜй»‘иүІпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮ ctx.strokeStyle = color жқҘдҝ®ж”№ stroke ж—¶иҝһзәҝзҡ„йўңиүІ
ctx.strokeStyle = "blue"; // дҝ®ж”№иҝһзәҝйўңиүІ
ctx.lineTo(300, 300);
ctx.stroke();

еҹәдәҺд»Јз Ғзҡ„йҖ»иҫ‘жҲ‘д»¬еҲҶжһҗжңҖеҗҺдёүиЎҢпјҡ

  1. дҝ®ж”№ strokeStyleпјҢйӮЈд№Ҳд»ҘеҗҺ stroke ж–№жі•з»ҳеҲ¶еҮәжқҘзҡ„зәҝжқЎдёәи“қиүІ
  2. ctx.lineTo(300, 300);еҚід»Һ (150, 150) дёҺ (300, 300) д№Ӣй—ҙвҖңжһ„жҖқвҖқдёҖжқЎзәҝ
  3. и°ғз”Ё stroke ж–№жі•з»ҳеҲ¶иҝҷжқЎзәҝпјҢйӮЈд№ҲжҲ‘д»¬зҡ„ canvas з”»еёғдёҠеә”иҜҘд»Һ (0, 0) еҲ° (150, 150) д№Ӣй—ҙдёәй»‘зәҝпјҢ(150, 150) еҲ° (300, 300) д№Ӣй—ҙдёәи“қзәҝ

дҪҶжҳҜе®һйҷ…дёҠз»ҳеҲ¶еҮәжқҘзҡ„з»“жһңжҳҜд»Һ (0, 0) еҲ° (300, 300) дёҖж•ҙжқЎзәҝйғҪжҳҜи“қиүІзҡ„гҖӮ

е…Ҳз»ҷз»“и®әпјҡ

з»“и®ә вҖӢ

canvas дёӯзҡ„з»ҳеҲ¶ж–№жі•пјҲвҖңеҠЁз¬” apiвҖқпјүйғҪд»ҘдёҠдёҖж¬Ў beginPath д№ӢеҗҺзҡ„жүҖжңүвҖңжһ„жҖқвҖқзҡ„и·Ҝеҫ„дёәеҹәзЎҖиҝӣиЎҢз»ҳеҲ¶гҖӮ

еҹәдәҺз»“и®әи§ЈйҮҠдёҖдёӢдёҠйқўзҡ„д»Јз ҒпјҡжҲ‘д»¬еҲӣе»әдёҠдёӢж–Үд№ӢеҗҺй»ҳи®Өе°ұи°ғз”ЁдәҶдёҖж¬Ў ctx.beginPath жқҘејҖеҗҜдёҖдёӘж–°и·Ҝеҫ„пјҢжүҖд»ҘдёҠйқўжІЎеҶҷпјҢдҪҶжҲ‘д»¬жҳҺзЎ®дёҠйқўзҡ„жүҖжңү canvas д»Јз ҒжҳҜеӨ„дәҺеҗҢдёҖдёӘ beginPath дёӯзҡ„пјҢдёҠйқўд»Јз Ғдёӯз¬¬дәҢж¬Ў stroke ж–№жі•и°ғз”Ёж—¶пјҢжҲ‘д»¬жҳҜеҹәдәҺдёҠйқўзҡ„жүҖжңүвҖңжһ„жҖқ apiвҖқдә§з”ҹзҡ„и·Ҝеҫ„зҡ„гҖӮд№ӢжүҖд»ҘжҲ‘д»¬зңӢеҲ°дёҖж•ҙжқЎи“қиүІзҡ„зәҝжҳҜеӣ дёәз¬¬дёҖдёӘ stroke ж–№жі•з»ҳеҲ¶зҡ„й»‘иүІзәҝжқЎиў«з¬¬дәҢж¬Ў stroke ж–№жі•з»ҳеҲ¶зҡ„и“қиүІзәҝжқЎиҰҶзӣ–дәҶпјҲз¬¬дәҢж¬Ў stroke ж–№жі•з»ҳеҲ¶ж—¶пјҢи·Ҝеҫ„дёӯе·Із»ҸжңүдәҶд»Һ (0, 0) еҲ° (300, 300) зҡ„зәҝпјүгҖӮ

еҰӮжһңжҲ‘д»¬жғіз»ҳеҲ¶ (0, 0) еҲ° (150, 150) зҡ„зәҝжқЎдёәй»‘иүІпјҢ(150, 150) еҲ° (300, 300) дёәи“қиүІпјҢеҸҜд»Ҙдҝ®ж”№д»Јз ҒеҰӮдёӢпјҡ

js
const ctx = canvasElm.getContext('2d');
ctx.moveTo(0, 0);
ctx.lineTo(150, 150);
ctx.stroke();
ctx.strokeStyle = "blue";
вҖӢ
ctx.beginPath(); // дёҠйқўз¬¬дёҖж¬Ў stroke з»ҳеҲ¶е®ҢжҲҗпјҢдёәдәҶи®©дёӢйқўзҡ„ stroke дёҚеҸ—дёҠйқўзҡ„вҖңжһ„жҖқвҖқзҡ„еҪұе“ҚпјҢи°ғз”Ё beginPath жқҘејҖеҗҜдёҖдёӘж–°зҡ„и·Ҝеҫ„
ctx.moveTo(150, 150); // еңЁдёҖдёӘж–°зҡ„ Path дёӯпјҢзӣёеҪ“дәҺжё…з©әдәҶжӣҫз»Ҹ Path дёӯзҡ„вҖңжһ„жҖқвҖқ(и·Ҝеҫ„)пјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒе…Ҳи°ғз”Ё moveTo е…Ҳз§»еҠЁеҲ°дёҖдёӘзӮ№пјҢжҚўеҸҘиҜқиҜҙ lineTo ж–№жі•йңҖиҰҒеүҚйқўжңүдёҖдёӘзӮ№дёәеҹәзЎҖпјҢдёҚз„¶ж— ж•Ҳ
ctx.lineTo(300, 300);
ctx.stroke(); // з»ҳеҲ¶еҮәжқҘдәҶд»Һ (150, 150) еҲ° (300, 300) зҡ„и“қиүІзәҝж®ө дё” (0, 0) еҲ° (150, 150) зҡ„й»‘иүІзәҝж®өе®үз„¶ж— жҒҷ

closePath дёҺ beginPath зҡ„е…ізі» вҖӢ

зӯ”пјҡжІЎжңүе…ізі»пјҢctx.closePath зӣёеҪ“дәҺдёҖдёӘвҖңжһ„жҖқ apiвҖқ пјҢеҹәдәҺдёҠйқўжҲ‘д»¬еҜ№дәҺ canvas з»ҳеҲ¶зҡ„зҗҶи§ЈдёҫдёӘдҫӢеӯҗпјҡ

js
ctx.moveTo(200, 0);
ctx.lineTo(200, 300);
ctx.lineTo(150, 150); // еҪ“дёӢе·Із»ҸвҖңжһ„жҖқвҖқдәҶдёӨжқЎзәҝ
ctx.closePath(); // closePath еҚіи®©еҪ“еүҚзҡ„жһ„жҖқзәҝжқЎиө·зӮ№дёҺз»ҲзӮ№зӣёиҝһпјҢеҪўжҲҗй—ӯеҗҲеҢәеҹҹ
ctx.stroke(); // з»ҳеҲ¶еҮәдёҖдёӘе°Ғй—ӯзҡ„дёүи§’еҪў

иҖҢ beginPath дҪңз”ЁжҳҜдҝқиҜҒжҲ‘д»¬дёӨдёӘдёҚеҗҢзҡ„з»ҳеҲ¶пјҲвҖңеҠЁз¬” apiвҖқпјүдёҚзӣёдә’еҪұе“ҚгҖӮ

зҗҶи§ЈдҪ•дёәвҖңеҹәдәҺзҠ¶жҖҒвҖқ вҖӢ

зҠ¶жҖҒзҡ„жҰӮеҝөжҲ‘д»¬е№¶дёҚйҷҢз”ҹпјҢvue дёҺ react иҝҷдәӣеүҚз«ҜжЎҶжһ¶зҡ„ж ёеҝғжҖқжғійғҪжҳҜвҖңзҠ¶жҖҒй©ұеҠЁи§ҶеӣҫвҖқпјҢиҝҷйҮҢзҡ„зҠ¶жҖҒеҸҜд»ҘзҗҶи§Јдёәж•°жҚ®пјӣйӮЈд№Ҳ canvas дёӯпјҢжҜҸдёҖдёӘ beginPath йғҪзӣёеҪ“дәҺејҖеҗҜдёҖдёӘж–°зҡ„и·Ҝеҫ„пјҢcanvas дёӯзҡ„зҠ¶жҖҒе®Ңе…ЁеҸҜд»ҘзҗҶи§Јдёәи·Ҝеҫ„дёӯжҲ‘д»¬вҖңжһ„жҖқвҖқзҡ„зәҝжқЎпјҢжңҖз»Ҳи°ғз”Ё stroke\fill зӯүз»ҳеҲ¶ж–№жі•ж—¶з»ҳеҲ¶еҮәжқҘзҡ„ж•ҲжһңжҳҜеҹәдәҺ lineTo зӯүж–№жі•жһ„йҖ еҮәжқҘзҡ„вҖңзҠ¶жҖҒвҖқзҡ„гҖӮ
